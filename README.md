# 康軒國語學習 App 安裝說明

## 📁 檔案清單

上傳到 GitHub 時，需要以下 4 個檔案：

```
index.html      ← 主程式（請勿改名）
manifest.json   ← App 設定檔
sw.js           ← 離線快取程式
icon-192.svg    ← App 圖示
icon-512.svg    ← App 圖示（大）
```

---

## 🚀 部署步驟（GitHub Pages）

### Step 1：建立 GitHub 帳號
前往 https://github.com 註冊免費帳號

### Step 2：新建 Repository
1. 登入後點右上角 **＋** → **New repository**
2. Repository name 填：`kangxuan-chinese`（或任何名稱）
3. 選 **Public**
4. 點 **Create repository**

### Step 3：上傳檔案
1. 點頁面中的 **uploading an existing file**
2. 把以下 5 個檔案全部拖進去：
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.svg`
   - `icon-512.svg`
3. 點 **Commit changes**

### Step 4：開啟 GitHub Pages
1. 進入 Repository → 點上方 **Settings**
2. 左側選單找 **Pages**
3. Source 選 **Deploy from a branch**
4. Branch 選 **main**，資料夾選 **/ (root)**
5. 點 **Save**
6. 等約 1~3 分鐘，頁面會出現你的網址：
   ```
   https://你的帳號.github.io/kangxuan-chinese/
   ```

---

## 📱 手機安裝方式

### iPhone / iPad（Safari）
1. Safari 開啟上面的網址
2. 點下方工具列的 **分享** 按鈕（□↑）
3. 往下找到 **「加入主畫面」**
4. 點 **新增**
5. App 圖示會出現在桌面！

### Android（Chrome）
1. Chrome 開啟網址
2. 點右上角 **⋮** 選單
3. 點 **「安裝應用程式」** 或 **「新增至主畫面」**
4. 點 **安裝**
5. App 圖示會出現在桌面！

## ✅ 功能特色
- 📖 一年級上下學期 1~12 課生字詞彙
- 🔊 發音朗讀（含注音，聲調居中）
- ✏️ 筆畫練習（可手寫）
- 🌿 部首學習（含部首注音）
- 🎮 闖關遊戲（滿分得 100 積分）
- 👤 個人檔案（名稱、性別、頭像）
- 🎁 積分兌換獎勵系統
- 📱 完全離線可用（資料存在手機）
